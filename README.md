### Instale as dependencias

- acesse a pasta /(Raiz) do projeto pelo seu terminal

- e digite o seguinte:

```
$ npm install
```

- para executar o servidor parcel, entre na pasta ```/src``` e digite:

```
$ npx parcel src/index.html
```
